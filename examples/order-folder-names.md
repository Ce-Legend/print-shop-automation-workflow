# Order Folder Names

这个系统靠文件夹名称识别订单规格。真实使用时，店员只需要把一单照片放进监控目录，系统会解析尺寸、模式、张数和订单号。

## 推荐格式

```text
【拍立得】5寸,10张 250701-123456789012345
【全景】6寸,8张 250702-123456789012346
3T_拍立得_5张250703-123456789012347
五寸拍立得,12张 250704-123456789012348
```

## 系统会提取什么

| 信息 | 例子 | 用途 |
|---|---|---|
| 尺寸 | 3寸、5寸、6寸、3T | 匹配纸张和预设 |
| 模式 | 拍立得、全景 | 决定是否预处理、是否套用对应预设 |
| 张数 | 10张 | 方便核对订单数量 |
| 订单号 | 250701-123456789012345 | 方便跟踪和归档 |

## 为什么要这样设计

打印店场景里，订单来源可能很散：聊天软件、网盘、手机相册、临时压缩包。与其让店员打开软件填表，不如把规则放进文件夹名。文件夹就是任务单，系统只负责解析和执行。
